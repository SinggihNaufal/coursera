# Simple Interest Calculator

## Description
This project is a simple Bash-based calculator that computes simple interest based on user input. It is designed to demonstrate basic scripting, user input handling, and arithmetic operations using Git and GitHub.

## Features
- Calculates simple interest
- Accepts user input for:
  - Principal amount
  - Interest rate
  - Time period
- Displays the calculated result

## Formula
Simple Interest is calculated using the formula:

SI = (P × R × T) / 100

Where:
- P = Principal amount
- R = Rate of interest
- T = Time period
